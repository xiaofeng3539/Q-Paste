Q-Paste
一款极简、高效的桌面端剪贴板管理工具，助你告别繁杂的复制粘贴。


🌟 核心功能
全局唤醒：随用随叫，彻底融入且不打断当前工作流。
极速响应：轻量级架构，保持极低的系统资源占用。
多格式支持：纯文本、链接、代码片段，一次复制，随时取用。
纯粹安全：数据仅做本地化存储，无需强制联网，断网亦可用。


📥 如何获取
请前往本仓库右侧的 Releases 页面，下载最新版本的安装程序：
文件名：Q-Paste Setup 1.0.0.exe
安装：下载后双击运行，按照向导完成安装即可。


🚀 快速开始
启动：打开 Q-Paste，程序将静默运行于系统托盘。
呼出：按下全局快捷键，即可在光标处唤出剪贴板历史列表。
上屏：使用鼠标点击或键盘上下键选择目标记录，回车即可将内容极速填入当前应用。
